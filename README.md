# 🌍 بازی کره زمین (The Earth Game)

ربات تلگرامی بازی استراتژیک متنی، نزدیک به حال‌وهوای Hearts of Iron IV، که در سال ۲۰۲۶ جریان دارد.
هر بازیکن رهبر یک کشور است و باید آن را در سه محور **اقتصاد، دیپلماسی و نظامی** مدیریت کند.
دستورهای آزاد بازیکن‌ها توسط هوش مصنوعی (**Groq** / مدل `gpt-oss-120b`) تفسیر و داوری می‌شوند.

---

## ✨ امکانات

- **کشورگیری** با تأیید مالک بازی (۳۶ کشور، ۷ کشور VIP)
- **اقتصاد**: گزارش اقتصادی، ذخایر استراتژیک، احداث معدن/کارخانه فولاد/سکوی نفت‌وگاز، فروش ذخایر
- **WTO**: ارسال محموله با زمان رسیدن تخمینی AI
- **دیپلماسی**: نامه، تماس تلفنی ۵ دقیقه‌ای (با لاگ مدیران)، دیدار حضوری، قرارداد، تحریم
- **نظامی**: پنل تجهیزات، ۴ نوع حمله (هوایی/زمینی/دریایی/خرابکارانه) با سنجش AI و مصرف سوخت
- **مشاور هوشمند**: در هر دامنه هر ۲۴ ساعت یک‌بار
- **کانال‌های خبری**: نظامی، دیپلماسی، اقتصادی، WTO
- معماری کاملاً **ماژولار** و کدِ انگلیسی با کامنت فارسی

---

## 🧱 معماری

```
bot/

```

## 🚀 راه‌اندازی

### ۱) پیش‌نیازها
- Python 3.11+ (تست‌شده روی 3.14)
- یک دیتابیس **PostgreSQL** فعال
- توکن ربات از [@BotFather](https://t.me/BotFather)
- کلید API از [Groq](https://console.groq.com)

### ۲) نصب وابستگی‌ها
```bash
pip install -r requirements.txt
```

### ۳) تنظیم فایل `.env`
فایل نمونه را کپی کنید و مقادیر را پر کنید:
```bash
cp .env.example .env
```
سپس در `.env`:
- `BOT_TOKEN` توکن ربات
- `DATABASE_URL` به شکل `postgresql+asyncpg://user:pass@localhost:5432/earthgame`
- `GROQ_API_KEY` کلید گرو
- `OWNER_IDS` / `ADMIN_IDS` آی‌دی عددی مالک و مدیران (با کاما)
- آی‌دی کانال‌ها و گروه‌ها (نظامی/دیپلماسی/اقتصاد/WTO/تماس/لاگ)

> ربات باید در همه‌ی کانال‌ها/گروه‌های خبری **ادمین** باشد تا بتواند پیام ارسال کند.

### ۴) آماده‌سازی دیتابیس (ساخت جدول‌ها + بارگذاری ۳۶ کشور)
```bash
python -m scripts.seed
```

### ۵) اجرای ربات
```bash
python -m bot
```

## 🎮 جریان بازی

1. بازیکن `/start` می‌زند و از «🌍 کشورگیری» یک کشور انتخاب می‌کند.
2. درخواست برای مالک ارسال می‌شود؛ مالک تأیید/رد می‌کند.
3. پس از تأیید، پنل مدیریت کشور با سه محور اصلی باز می‌شود.
4. بازیکن اقتصاد را رشد می‌دهد، با کشورها دیپلماسی می‌کند و در صورت لزوم دست به حمله می‌زند.

---

## 🔧 تنظیم سختی بازی

همه‌ی اعداد بازی (هزینه‌ی ساخت‌وسازها، بازدهی‌ها، کول‌داون‌ها، مدت تماس/دیدار) در
`bot/constants.py` متمرکز شده‌اند و از روی پلی‌بوک تنظیم شده‌اند.

شناسه‌ی مدل AI نیز در `.env` (متغیر `AI_MODEL`) قابل‌تغییر است.

## 📝 نکات

- لایه‌ی دیتابیس از SQLAlchemy 2.0 (async) + asyncpg استفاده می‌کند.
- کارهای زمان‌دار (بازدهی ذخایر، رسیدن محموله/سفر، پایان تماس/دیدار، اعلام نتیجه‌ی حمله)
  هر ۶۰ ثانیه توسط APScheduler پردازش می‌شوند.
- داده‌ی هر کشور پیش از ارسال به AI، از دیتابیس خوانده و به **JSON** تبدیل می‌شود
  (در `bot/services/ai/context.py`).
